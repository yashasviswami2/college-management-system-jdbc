# College Management System (Java JDBC + MySQL)

## 📌 Overview
This project is a backend implementation of a College Management System built using Java and JDBC with MySQL as the database.  
It follows the DAO (Data Access Object) design pattern and supports full CRUD operations.

## 🚀 Features
- Add, update, delete, and view students
- Course management
- Teacher management
- Subject management
- Marks recording system
- Attendance tracking
- Fee management

## 🛠 Technologies Used
- Java
- JDBC
- MySQL
- DAO Pattern
- IntelliJ IDEA

## 🗄 Database Setup
1. Open MySQL Workbench
2. Run the `database.sql` file
3. Update database credentials in `DBConnection.java`
4. Run the project

## 📂 Project Structure
- `src/` → Java source code
- `database.sql` → Database schema and sample data
- `README.md` → Project documentation

## 🎯 Learning Outcomes
- JDBC connectivity
- Foreign key relationships
- SQL constraints
- Modular backend structure using DAO
